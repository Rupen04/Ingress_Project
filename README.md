#_Ingress_Project_#
A complete project of microservices integrated and web page deployment using Docker, Kubernetes, Helm and Ingress Controller (Nginx)

 Learning Outcomes:-
 
 From this project, we learned: - Docker image creation & pushing to Docker Hub - Kubernetes Deployments,
 Services, Ingress - Writing & debugging YAML manifests - Installing Helm & NGINX Ingress Controller 
 Understanding LoadBalancer & ingress routing - Managing AWS EKS clusters - End-to-end microservices
 deployment on Kubernetes.
